# Rite

A CLI for starting new projects, the way you want.

# Features
- Template-based
- Configurable
- Extensible with plugins

# Getting Started

## Prerequisites
- NodeJS
- pnpm or yarn or npm

## Installation
`
# install it globally
# npm
$ npm install --g rite

# yarn
$ yarn global add rite

# pnpm
$ pnpm add -g rite
`

## Quick Start
Create a new project from a template
`
rite new [project-name]
`
